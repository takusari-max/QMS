================================================================================
KEY FORM STRUCTURE PATTERNS
================================================================================

Document Headers:
  - Row 2-3: Title (BOLD, SIZE:20.0, background color FFD9D9D9)
  - Row 4-7: Basic info fields (Customer, Subject, Date, Reviewer)
  - Column Q-R: Navigation links (Size:10.0, BOLD) and revision history

Common Elements:
  - Checkbox patterns: □ Yes □ No (SIZE:10-11pt, plain)
  - Text fields: Label in column B/C, merged cells E-M for input
  - Tables: Multiple merged cells creating grid structure
  - Instructions: SIZE:7-9pt, normal or bold weight
  - Required fields: Marked with "・" or shown in BOLD

Merged Cells:
  - Create visual form structure and field span
  - B6:D6, B7:D7, etc. = label area
  - E6:M6, E7:M7, etc. = input area
  - F20:AI21 = large input area for longer content

Color Codes:
  - FFD9D9D9 = light gray header background
  - Other colors: used for specific sections or emphasis
